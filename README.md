# Duratex Test

## Installation

1. Clone repository: `git clone https://github.com/RafaelAugustoS/Duratex-Test.git`
2. Run (in folder): `npm i`
3. Run: `react-native link`
4. Run: `react-native run-ios` or `react-native run-android`
